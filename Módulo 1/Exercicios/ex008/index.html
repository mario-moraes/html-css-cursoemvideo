Negrito / Destaque

Nesta frase temos um termo em destaque utilizando a tag b (não semântica).

Nesta frase, utilizamos a versão semântica da tag chamada de strong.

Nesta frase, colocamos o termo em itálico usando a tag "i" (não semântica)

Nesta frase temos um termo em ênfase usando a tag "em" (semântica)

Por fim, para dar uma enfase ainda maior com efeito de marcador, utilize a tag "mark" para realçar (com significado semantico) o seu texto! Aqui a cor foi modificada com o comando "style", algo de formatação CSS.

E no outro paragrafo, outro exemplo de texto marcado, aqui a cor esta padronizada porque se utillizou a tag mark com uma formatação CSS no head do código

Texto Grande e Texto Pequeno

Estamos criando aqui um exemplo de texto grande e um exemplo de texto pequeno

Texto deletado

Podemos marcar um texto deletado, envelopando ele com a tag "del" para indicar que ele deve ser lido mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase com a tag semantica "ins"e indicar que ele foi adicionado depois ou sublinhado

Texto sobrescrito

Utilizado bastante em fórmulas matemáticas, usa-se a tag x20+3, envelopado com tag "sup"

Texto subscrito

Para inserir coisas como fórmulas químicas, exemplo de H20, envelopando o caractere com a tag "sub"
