Código-fonte / Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript, envelope o termo com a tag "code".
Para que o código fique indentado, envelope a tag "code" e o respectivo código na tag "pre", pois o <pre> irá considerar as quebras, logo, por exemplo:

        
            num = int(input('Digite um número))
            if num % 2 == 0:
                print(f'0 número {num} é PAR')
            else:
                print(f'0 número {num} é ÍMPAR')
            print('Fim do Programa')
