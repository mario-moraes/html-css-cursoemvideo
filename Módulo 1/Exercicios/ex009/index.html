Listas ordernadas

A tag "ol" possui parâmetro type, onde configuramos o tipo de marcador da lista atual tais como: 

            1 - valor padrão. Cria lista numeradas "1,2,3..."
            A - Cria listas alfabéticas maiusculas. "A, B, C..."
            a - Cria listas alfabéticas minusculas. "a,b,c..."
            I - Cria listas com algarismo romanos maiusculos. "I, II, III..."
            i - Cria listas com algarismos romanos minusculos. "i, ii, iii..."
Pode ainda indicar o inicio da contagem a partir de um numero especifico usando o parâmetro "start", exemplo de linha de código: type="I" start="5">

Lista de Definições

São como se fossem "dicionarios", é uma lista sem demarcadores mas bem útil em alguns casos. Estão dentro de tags "dl" definition list, "dt" definition term, "dd" definition description.

HTML
Linguagem de marcação utilizada para criar conteúdo em sites.
CSS
Linguagem de marcação utilizada para especificação de estilos.
Javascript
Linguagem de programação para criar interatividade.
