Aprendendo a usar o embed (incorporação)

O jeito mais prático para economizar banda para o cliente, é utilizar outros hospedadores de video, como o próprio youtube. Tal prática inclusive auxilia na visibilidade do site através do algoritmo de busca do Google.

Vale ressaltar que essa prática é para videos públicos. Para serviço de assinantes utilizar um servidor que ofereça opção privada de embed do video, como o Vimeo, para cursos por exemplo.
